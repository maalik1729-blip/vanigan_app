# Assets Folder

This folder contains app assets like icons and splash screens.

## Required Assets

For now, we'll use Expo's default assets. You can replace these later with your custom designs.

## Asset Specifications

- **icon.png**: 1024x1024 px (app icon)
- **splash.png**: 1284x2778 px (splash screen)
- **adaptive-icon.png**: 1024x1024 px (Android adaptive icon)
- **favicon.png**: 48x48 px (web favicon)
